# 📊 Sales Performance Dashboard (Power BI)

## Overview

This project presents an interactive **Sales Performance Dashboard** built using **Power BI** to analyze e-commerce sales data and uncover actionable business insights. The dashboard tracks key metrics such as sales, profit, quantity sold, customer behavior, product performance, and payment preferences.

Designed as an end-to-end analytics project, it combines **data cleaning, modeling, DAX measures, and interactive visualizations** to support data-driven decision-making.

---

## 🚀 Project Objective

The goal of this project is to:

* Analyze overall sales performance across regions, customers, and product categories.
* Identify profitable products and low-performing areas.
* Monitor monthly profit trends and seasonal patterns.
* Understand customer purchasing behavior and payment preferences.
* Build an interactive dashboard for business stakeholders.

---

## 🛠️ Tech Stack

* **Power BI** – Dashboard creation and data visualization
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – KPI calculations and measures
* **CSV Dataset** – Source data for orders and transaction details

---

## 📂 Dataset Information

The project uses two datasets:

### Orders Dataset

Contains customer and order-level information:

* Order ID
* Order Date
* Customer Name
* State
* City

### Details Dataset

Contains transaction-level sales data:

* Order ID
* Sales Amount
* Profit
* Quantity
* Category
* Sub-Category
* Payment Mode
